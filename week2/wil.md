# 2주차 정리
## git clone (코드 복사)
- Github에 들어가서 Repository Code의 URL을 복사
- git clone <URL> : github의 레포지토리 코드를 컴퓨터로 가져오기 (원하는 폴더에 복제 가능)
### 다시 열기
- ls : 지금 위치에 어떤 파일이 있는지 확인
- cd <폴더명>: 지금 위치에서 폴더 위치를 변경
- cd .. : 폴더 밖으로

## 페이지 연결하기 (a 태그)
` <p>엄마, <a href="./father.html">아빠</a>,저, 동생. 그리고 흰둥이와 함께 살아요.</p> `
- 아빠 페이지에 링크를 생성, 현재 페이지와 father.html 페이지를 연결한다.
### 돌아가기
- index.html -> python.html 한 다음에 다시 python.html -> index.html 하려면 `..`/index.html을 써야한다. 

## 이론
- 목차 페이지는 거미줄 형태로 연결된 지식의 체계에서 '첫 시작점'이다. 따라서 0쪽처럼 `'index.html 페이지'`는 생략된다.
- google.com = google.com/index.html

## 스타일 입히기
` <h1 style="color: yellow; background-color: red;">신짱구</h1> `
- color는 폰트의 색, background-color는 배경색.
- 인라인 스타일(inline): 태그에 직접적으로 스타일을 입히는 방식. 태그에 style이라는 속성을 통하면 구현할 수 있다.
### CSS (Cascading Style Sheets)
HTML 등의 마크업 언어로 작성된 문서가 실제로 웹사이트에 표현되는 방법을 정해주는 스타일 시트 언어. HTML에 `스타일`을 표현하게 해주는 언어.
